Functional languages with static typing like Standard ML and OCaml.

For Erlang VM

  • Alpaca
  • Gleam
  • Elchemy

For Java VM

  • Scala
  • Kotlin
  • Eta
  • Frege
  • Lux
  • Flix

For .net Runtime

  • F#

Transpiles to Javascript

  • BuckleScript
  • Fable
  • PureScript

Rest

  • Amulet transpiles to Lua
  • Coconut transpiles to Python
  • LunarML transpiles to Lua or Javascript
  • Roc compiles to machine code or WebAssembly. I find it interesting how it separates platforms and applications. Could we have a platform that targets microcontrollers?
